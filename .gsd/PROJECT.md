# PROJECT: Halo Bus Tracking

## Vision
Transforming the existing "Halo Bus" / "Halo Bus" / "Halo Bus" bus tracking application into a premium, professional product named **Halo Bus**. This involves a complete rebranding of the app name, package names, module names, and all internal references to the previous identities.

## Tech Stack
- **Mobile**: Flutter (Dart)
- **Frontend**: Web (Modern JS/HTML/CSS)
- **Backend**: Firebase (Auth, Firestore, Cloud Functions)
- **Realtime**: Node.js/Socket.io (Realtime Relay)

## Constraints
- All package names must change from `com.halobus.*` to `com.halobus.*`.
- Application names must change to "Halo Bus".
- Search and replace all instances of "Halo Bus", "Halo Bus", "Halo Bus" with appropriate "Halo Bus" equivalents.
- Ensure Firebase configuration (google-services.json, firebase_options.dart) is updated or flagged for user update since bundle IDs are changing.
